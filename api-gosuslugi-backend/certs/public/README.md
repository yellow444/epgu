# public/ — публичные CA и корневые сертификаты

Здесь хранятся **публичные** сертификаты, необходимые для работы приложения. Они безопасны для коммита в репозиторий.

| Файл | Назначение |
|---|---|
| `rootca.cer` | Корневой УЦ Минцифры |
| `ca-root.crt` | Дополнительный корневой УЦ |
| `test_ca_rtk2.cer` | Тестовый УЦ ЕСИА (RTK2) |
| `cacerts.p7b` | Пакет промежуточных CA |
| `der.cer` | Промежуточный CA |
| `2F0CB09BE3550EF17EC4F29C90ABD18BFCAAD63A.cer` | Промежуточный CA (по thumbprint) |

Эти сертификаты устанавливаются в хранилища `root` / `mroot` на старте контейнера (см. `../entrypoint test.sh`).

> Сертификат `test_ca_rtk3.cer` для основного production-образа скачивается из `certenroll.test.gosuslugi.ru` во время сборки (см. `Dockerfile`) и в репозиторий не коммитится.

Личные сертификаты пользователей — в [../personal/](../personal/README.md).
